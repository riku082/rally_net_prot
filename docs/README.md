## ドキュメントの追加

新しいドキュメントを追加する際は、以下のカテゴリに従って適切なディレクトリに配置してください：

1. **開発関連** → `development/`
   - 開発ログ、進捗レポート、コードレビューなど

2. **仕様書** → `specifications/`
   - 機能仕様、アーキテクチャ設計、技術仕様など

3. **ガイド** → `guides/`
   - セットアップ手順、操作マニュアル、ベストプラクティスなど
